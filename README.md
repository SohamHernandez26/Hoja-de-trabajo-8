# Hoja-de-trabajo-8
Gabriel Zambrano, Zabdi Pelicó

Este proyecto simula la gestión de procesos en un sistema operativo moderno, comparando el rendimiento de diferentes estructuras de datos utilizadas para organizar procesos según su tiempo de ejecución virtual (vruntime).

Se implementan y analizan tres tipos de árboles:

Árbol Binario de Búsqueda (BST)
Splay Tree
Red-Black Tree

El objetivo es evaluar la eficiencia de estas estructuras en distintos escenarios de carga, similar a cómo lo hace el planificador de procesos del sistema operativo Linux.
